Projeto feito com o intuíto de imprimir um boletim utilizando o JasperReport, ele faz uso de:

-InteliJ
-Java 8
-SQL Server Management Studio
-Docker
-Visual Studio Code
-AngularJs 1.6.9
-NodeJs

Depois de baixar e instalar tudo corretamente, vá ao terminal do InteliJ e digite docker-compose up.
Cadastre primeiro o aluno depos a disciplina e por último as notas e só então imprima o boletim.

Cuidados ao uso:
Não cadastre nomes nem disciplinas muito grandes pois pode prejudiar a estética no front.
Para cadastrar notas utilize . para separar o decimal, Ex: 8.75
